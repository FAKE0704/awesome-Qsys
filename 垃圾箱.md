# 数据流程
第i日
已经收集了所有信号和数据
准备决定在第i日进行操作
根据第i-1日的信号、仓位、余额、购买情况
决定在第i日做什么操作，将以第i日的价格进行结算（类似滑点）
止盈止损将以第 i-日的价格进行结算（因为是提前设定的）  # 。。。


# pairTrade逻辑
- 有买入信号
  - 未买入
    - 买
  - 已买入
    - 距上次买入已跌超3%
      - 买
    - 未超3%
      - 不买
- 无买入信号
  - 不买
# conditionTrade
## buy
- 已买入
  - 不买
- 未买入
  - 有买入信号
## sell
- 买入时间还在当天 |未买入
  - 不卖
- 买入时间超过n天 | 止损 | 已盈利6% | (有卖出信号 & 盈利>3%)
  - 卖
