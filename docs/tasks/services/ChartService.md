# ChartService

## Responsibilities
- 图表服务管理

## Configuration Classes
- ChartConfig: 图表配置
- LayoutConfig: 布局配置
- DataConfig: 数据配置

## Chart Classes
- ChartBase: 图表基类
- CandlestickChart: K线图
- VolumeChart: 成交量图
- CapitalFlowChart: 资金流向图

## Indicator System
- Indicator: 指标基类
- 具体指标实现

## Design Patterns
- ChartFactory: 图表工厂模式
- IndicatorFactory: 指标工厂模式
- IndicatorDecorator: 指标装饰器模式

## Data Containers
- DataBundle: 数据容器
